# 期权数据
介绍：可以获取期权tick数据、greeks数据、分时数据、k线数据、可以查询指定月份和时间的看涨和看跌期权，
可以获取股票tick行情

### 功能
导入模块
```python
from apps.option_data import *

```

* 1、获取合约名称
```python
test_contract_name = contract_name('10001011')
print(test_contract_name)
```
```
输出：
50ETF购11月2600
```

    test_contract_monthly = contract_monthly('300ETF')
